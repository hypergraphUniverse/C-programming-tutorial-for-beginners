mission_3.1问答

1. 
声明了一个数组"int a[10];"
请问这个数组中角标最大的和最小的变量分别是：a[9],a[0]。

在给这个数组中的变量赋值的时候，使用到了语句"a[2]=3;"。
请解释，上述语句中的a的变量类型是：int*类型变量/int指针类型变量。
请解释，其中的中括号"[]"是一种：操作符
a[2]的意义是：指针a指向的地址向后偏移两个地址的int类型的变量。

2. 
数组声明并初始化的时候，出现了如下几种情况。请判断这个声明是否正确，如果正确，请说出这个数组中一共有几个变量，值是多少（初始值未知回答未知即可，忽略变长数组特性）

int a[5];
答案：正确，有五个变量，初始值未知。

int a[5]={1,2,3};
答案：正确，有五个变量，初始值分别为1，2，3，0，0。

int a[];
答案：错误。

int c;
scanf("%d",&c);
int a[c];
答案：错误。

int a[]={1,2,3};
答案：正确，有三个变量，初始值分别为1，2，3。
